
# QRPhishing

Hijacking the google whatsapp web session using QRPhishing
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Chrome Driver Must be in your Path

Start the server

```bash
  npm run start
```
